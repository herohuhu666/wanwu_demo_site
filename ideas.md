# 万物 App 互动演示网站 - 设计构思

## 设计目标
构建一个高保真的互动演示网站，完美复刻万物 App 的极简主义美学，同时增加适合 Web 端展示的引导和说明元素，让投资人能够沉浸式体验产品功能。

## 核心理念
**"Digital Zen" (数字禅意)**
- **极简留白**：大面积的留白，呼吸感强的布局。
- **克制交互**：微动效，不打扰用户的注意力。
- **自然色彩**：以黑白灰为主，辅以低饱和度的功能色（绿、紫、橙）。
- **衬线字体**：使用优雅的衬线字体（如 Playfair Display 或 Noto Serif）搭配现代无衬线字体，营造高端、静谧的氛围。

## 视觉风格
- **背景**：柔和的米白色或极浅的灰色，模拟纸张质感。
- **卡片**：轻微的阴影，圆角适中，悬浮感。
- **图标**：细线条，极简风格。
- **动效**：缓慢、流畅的淡入淡出，模拟水墨晕染或纸张翻页。

## 页面结构
1. **引导页 (Landing)**：
   - 极简的 Hero Section，展示 App 核心价值。
   - "开始体验" 按钮，平滑滚动到模拟器区域。
2. **模拟器区域 (Interactive Demo)**：
   - 左侧/中央：手机外壳包裹的互动区域，运行 App 的 Web 版复刻。
   - 右侧/浮层：功能说明和引导提示（根据当前操作动态变化）。
3. **功能模块**：
   - **签到**：点击签到，水波纹扩散效果。
   - **灵犀**：卡片翻转或淡入显示灵感文案。
   - **状态**：滑动选择器或极简按钮组。
   - **会员**：优雅的对比表，开关切换时的色彩流转。

## 字体系统
- **标题**：Playfair Display (Google Fonts) - 优雅、经典。
- **正文**：Inter 或 Lato (Google Fonts) - 清晰、现代。

## 色彩系统
- **Primary**: #1a1a1a (接近黑色，用于主要文字和按钮)
- **Background**: #f9f9f7 (米白色，纸张质感)
- **Accent Green**: #4a7c59 (低饱和度绿，用于签到成功)
- **Accent Purple**: #7d5a8a (低饱和度紫，用于灵犀)
- **Accent Orange**: #d48c46 (低饱和度橙，用于会员)
